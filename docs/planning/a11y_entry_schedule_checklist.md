# エントリ作物スケジュール — アクセシビリティ確認（NFR-04）

**対象**: 一覧テーブル、適否表示、4フェーズ一覧、エラーメッセージ。

## チェックリスト

1. **色だけに依存しない**: 適否は「適期あり／要確認」テキスト（`.eligible-pill`）を併記。
2. **操作可能なコントロール**: 農場セレクトにラベル（`sr-only`）、展開ボタンに `aria-expanded`。
3. **状態メッセージ**: 予測鮮度は `role="status"`、注記は `aria-label` 付き `region`。
4. **コントラスト**: 主要テキストと背景のコントラストを目視（WCAG 詳細数値は別ツールで確認）。
