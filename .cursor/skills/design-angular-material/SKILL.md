# Angular Material デザイン（一貫性担保）

Angular MaterialとHTML/CSSで一貫したUIデザインを実現するスキル。

## When to Use

- UIデザインの一貫性を確保したいとき
- Angular Materialのコンポーネント選定・実装時
- HTML構造・アクセシビリティ対応時
- Trigger: UIデザイン, 見た目の統一, Angular Material, テーマ, コンポーネント選定, HTML構造

## Instructions

- デザイントークンを用いてハードコードを避ける
- カードを基調としたデザインを適用

- For templates and samples, see [references/SAMPLES.md](references/SAMPLES.md)
