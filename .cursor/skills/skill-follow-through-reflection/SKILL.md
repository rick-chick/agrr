# スキル未遵守の振り返り

スキルの指示が守られなかったとき、構造分析により原因を特定し、改善につなげるスキル。

## When to Use

- スキルが守られていないと指摘されたとき
- 自分でスキル遵守を怠ったことに気づいたとき
- Trigger: スキル未遵守, 振り返り, 構造分析, スキル改善

## Instructions

- 事実を特定し、観点別分析（手順・解釈・優先度・ルール処理の 4 観点）を行う
- 振り返り結果をスキル改善に反映
- For detailed reflection process, see [references/REFLECTION_PROCESS.md](references/REFLECTION_PROCESS.md)
- For example analysis, see [references/EXAMPLE.md](references/EXAMPLE.md)
